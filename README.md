# Laravel Instagram Clone

## 概要

Laravelを用いて開発した、Instagram風のSNSアプリケーションです。
ユーザー同士のつながりや投稿機能を中心に、基本的なSNS機能を実装しました。

## 開発背景

Webアプリケーション開発の理解を深めるため、ユーザー認証やデータベース設計を含む実践的な開発として制作しました。

## 開発環境

* PHP
* Laravel
* MySQL
* HTML / CSS
* JavaScript
* Git / GitHub

## 主な機能

* ユーザー登録 / ログイン / ログアウト
* 投稿機能（作成 / 編集 / 削除）
* コメント機能
* いいね機能
* フォロー / フォロワー機能
* プロフィール編集
* ユーザー検索機能
* 管理者機能

## 担当・実装内容

本プロジェクトは個人で開発を行い、以下の機能を実装しました。

* Laravelの認証機能を用いたログインシステム
* 投稿データのCRUD処理
* ユーザー間のフォロー関係の管理
* いいね・コメント機能の実装
* データベース設計およびテーブル構築

## 工夫した点

ユーザー同士の関係性（フォロー・フォロワー）や投稿に対するリアクション（いいね・コメント）を実装し、SNSとしての基本的な機能を網羅しました。

また、MVCアーキテクチャを意識して開発を行い、コードの可読性と保守性を意識しました。

## データベース設計

ユーザー、投稿、コメント、いいね、フォロー関係などのテーブルを設計し、それぞれの関連性を意識して構築しました。

## セットアップ方法

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
